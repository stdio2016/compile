pscanner

這個程式可以掃描 P 語言程式的 token，並把它列出來
特點
1. 字串中的兩個雙引號會轉成 1 個雙引號，讓字串的內容更容易了解
2. 列出 token 的時候也會顯示原始程式
3. 可以用 //&S- 關掉顯示程式碼功能， //&T- 關掉列出 token 功能
4. 遇到無效字元，會顯示錯誤訊息，並結束本程式
5. 會檢查八進位整數 (0 開頭的整數) 是否合法
6. 會檢查 /* ... */ 註解是否完整
7. 規格上所說的 token 都有做

執行平台
目前只有 Ubuntu Linux

怎麼執行
請先安裝 flex
打開這個資料夾，然後輸入命令列 make
就會看到一個檔案叫做 pscanner
在命令列輸入 ./pscanner <檔案名> 就可以執行
如果你手邊沒有 P 程式，可以試試這個：
    ./pscanner test/hello.txt
test 資料夾裡的檔案是測試資料
